# Simon Game
Course EE296 (Sophmore Projects)

## Background
Simple simon game (Arduino memory game)

LED will light up in random pattern, Player needs to memorize pattern and push button to match pattern.


## Equiptment
2 Buttons,
2 Light Emitting Diodes,
1 Buzzer/speaker,
2 1k resistors,
1 Arduino uno,
9 wires.
